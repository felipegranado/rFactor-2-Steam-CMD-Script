# rFactor-2-Steam-CMD-Script

### PT
Script para fazer download dos conteúdos da Oficina da Steam e fazer a instalação automática. Perfeito para servidores que possuem muitos conteúdos a serem baixados e instalados.

Passos:
1. Clique no arquivo .bat;
2. Insira os IDs dos conteúdos da Oficina Steam, separados por vírgula;
3. Escolha se quer manter todos os arquivos, manter apenas os arquivos .RFCMP e .RFMOD ou se quer deletar tudo;
4. Veja a mágica!

### EN
Script to download the contents of the Steam Workshop and make the automatic installation. Perfect for servers that have a lot of content to download and install.

Steps:
1. Click on the .bat file;
2. Enter the Steam Workshop content IDs, separated by comma;
3. Choose whether you want to keep all files, keep only the .RFCMP and .RFMOD files, or delete all files;
4. See the magic!
